--- title: 'GlassFish v3 FishCAT Announced' author: 'Jason Lee' layout: blog-post date: 2009-9-23 tags: [ GlassFish ] --- During the GlassFish v3 Prelude development cycle, the GlassFish team launched an initiative called FishCAT, which is our Community Acceptance Testing program. The program was very successful for Prelude, resulting in many, many reported (and fixed! : ) issues for the Prelude release. As we press hard toward the release of GlassFish v3 final later this fall, this program has been re-launched to help us engage our user community. Through this program, testers will have a much greater impact on the final quality of the product when it finally ships. If you'd like more information, you can see the announcement here, and a description of the program here. If you'd like to sign up and help, the application can be found here.
